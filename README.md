# MAXON Marketing Home Page

A modern, responsive marketing website for MAXON's 3D creation software, featuring Cinema 4D and other professional tools.

## 🌟 Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Performance Optimized**: Fast loading with lazy loading and throttled scroll events
- **Accessibility**: Keyboard navigation support and semantic HTML
- **SEO Friendly**: Proper meta tags and structured content
- **Interactive Elements**: Smooth scrolling, animated counters, and hover effects

## 📁 Project Structure

```
/
├── index.html          # Main HTML file
├── styles.css          # CSS styles and responsive design
├── script.js           # JavaScript functionality
├── Copyrights.html     # Legal information (existing)
└── README.md           # This documentation
```

## 🚀 Getting Started

1. **Clone or download** the repository
2. **Open `index.html`** in a web browser
3. **No build process required** - it's a static website

### Local Development

For the best development experience, serve the files through a local server:

```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (with http-server)
npx http-server

# Using PHP
php -S localhost:8000
```

## 🎨 Design System

### Colors
- **Primary**: `#FF6B35` (MAXON Orange)
- **Secondary**: `#F7931E` (Gradient Orange)
- **Dark**: `#1a1a1a` (Text/Backgrounds)
- **Light**: `#f8f9fa` (Backgrounds)
- **Gray**: `#666` (Secondary text)

### Typography
- **Font Family**: Inter (Google Fonts)
- **Weights**: 300, 400, 500, 600, 700

### Breakpoints
- **Mobile**: `480px` and below
- **Tablet**: `768px` and below
- **Desktop**: `1200px` max-width

## 🔧 Customization

### Updating Content
1. **Hero Section**: Edit the title and description in `index.html`
2. **Features**: Modify the features grid with new icons and descriptions
3. **About Section**: Update company information and statistics
4. **Footer**: Add or modify links and social media

### Styling Changes
1. **Colors**: Update CSS custom properties in `styles.css`
2. **Typography**: Modify font imports and sizes
3. **Layout**: Adjust grid systems and spacing
4. **Animations**: Customize keyframes and transitions

### Adding Functionality
1. **Form Integration**: Add contact forms with validation
2. **Analytics**: Integrate Google Analytics or similar
3. **CMS**: Connect to headless CMS for content management
4. **Animations**: Add more sophisticated animations with libraries
